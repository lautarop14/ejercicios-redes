Declaraciones en bloque definidas entre etiquetas style

h1 {
    font-size: 2em;
    color: yellow;
    text-align: center;
    background-color: blue;
}

Todos los textos con etiquetas h1 tienen este estilo.

La etiqueta h1 lleva texto en color amarillo.

El background de un h1 lleva color azul (como bokita)

El estilo definido para la etiqueta h1 indica que el texto irá centrado (text-align).

        p {
            text-align: center;
            font-size: 1em;
            color: blue;
            font-style: italic;
            border-top: 1px solid blue;
            border-bottom: 1px solid blue;
        }

Todos los textos con etiqueta p tienen este estilo definido dentro de la etiqueta style.

Para los párrafos se definió un estilo con estas características: alineación a la izquierda.

Tamaño de letra de la misma medida que la definida para el contenedor body.

Color de letra negra.

Estilo de la fuente itálico.

Borde superior e inferior sólido, de 2px y de color azul.
